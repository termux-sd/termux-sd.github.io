credmap

هذا الاسكربت وظيفته انه يعمل على معرفة من دخل بالايميل بتاعك في بعض المواقع هو اسكربت رائع يعطيك خريطة عن تسجيل دخول بالايميل الذي تملكه ما عليك الا ان تكتب الايميل الذي تريد معرفة دخوله في اي مواقع + الاسم الحساب للايميل + تعطي كلمة سر  ما مهم الا تكون كلمة السر بتاعت الحساب  اي كلمة سر وهمية 

طريقة التنصيب 
$pkg install git 
$pkg install python2
$pkg installl python 

$git clone https://github.com/lightos/credmap

 cd بعد يتم تنزيل الاسكربت الان يمكنك الانتقال الى مسار الاسكربت  بالامر اظن بعض الناس تعرفه

$cd  credmap 
الان نقوم باعطاء جميع الملفات صلاحية النظام وليس المكتبة المعينة التي يعمل بها الاسكربت

$chmod +x *

   تفسير عن  النجمة * هذه النجمة هي انك تقوم  باعطاء تصاريح الى كل الملفات والمجلدات الموجودة في المسار 
 chmod تفسير عن الامر  
  change mode  يقوم بعمل  
للملفات والمجلدات  
 umask يتم فلترتها عبر 

سوف اقوم لكم لاحقا في  قسم خاص بشرح الاوامر 

الان نقوم بتشغيل الاسكربت عذرا لقد خرجنا عن الشرح لكن لكي ازيد لكم المعرفة اكتر 

$python2 credmap.py -h 
هنا يقوم خيار -h 
يعطيك طريقة عمل الاسكربت 

مثال 

$python2 credmap.py  --email ali123@gmail.com --user ali ahmed 

  ali123@gmail.com  انت ما عليك سواء استبدال  
ali ahmed  وايضا
